\# Variáveis e Tipos de Dados



\## Variáveis

Uma variável é um espaço na memória do computador usado para armazenar um valor. Esse valor pode ser alterado ao longo da execução do programa.



\## Tipos de dados

Tipos de dados definem o tipo de valor que uma variável pode armazenar.



\## Principais tipos de dados



\### Inteiro (int)

Representa números inteiros, positivos ou negativos.

Exemplo: 10, -5, 0



\### Decimal (float)

Representa números com casas decimais.

Exemplo: 3.14, -2.5



\### Texto (string)

Representa sequências de caracteres.

Exemplo: "Olá", "Liz"



\### Booleano (bool)

Representa valores lógicos.

Exemplo: true, false
