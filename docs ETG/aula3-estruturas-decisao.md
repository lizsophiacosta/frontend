\# Estruturas de Decisão



\## O que são estruturas de decisão

Estruturas de decisão são usadas na programação para permitir que o programa tome decisões com base em condições.



\## Como funcionam

O programa avalia uma condição. Se ela for verdadeira, executa um bloco de código; caso contrário, pode executar outro.



\## Principais tipos



\### if

Executa um bloco de código se a condição for verdadeira.



Exemplo:

if (idade >= 18) {

&#x20;   // pode entrar

}



\### if-else

Permite escolher entre dois caminhos.



Exemplo:

if (idade >= 18) {

&#x20;   // maior de idade

} else {

&#x20;   // menor de idade

}



\### else if

Permite verificar múltiplas condições.



Exemplo:

if (nota >= 7) {

&#x20;   // aprovado

} else if (nota >= 5) {

&#x20;   // recuperação

} else {

&#x20;   // reprovado

}



\## Operadores de comparação

São usados para criar condições.



\- == igual

\- != diferente

\- > maior que

\- < menor que

\- >= maior ou igual

\- <= menor ou igual



\## Operadores lógicos



\- \&\& (E): todas as condições devem ser verdadeiras

\- || (OU): pelo menos uma condição deve ser verdadeira

\- ! (NÃO): inverte o valor lógico



\## Resumo

Estruturas de decisão permitem que o programa escolha diferentes caminhos com base em condições.
