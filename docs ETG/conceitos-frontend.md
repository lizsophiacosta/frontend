\# Frontend



\## O que é Frontend

Frontend é a parte de um sistema ou site com a qual o usuário interage diretamente. É tudo aquilo que aparece na tela.



\## Principais características

O frontend é responsável pela aparência, layout e interatividade de uma aplicação.



\## Tecnologias utilizadas



\### HTML

Responsável pela estrutura da página.



\### CSS

Responsável pelo estilo visual, como cores, fontes e layout.



\### JavaScript

Responsável pela interatividade, como cliques, animações e ações dinâmicas.



\## Exemplo

Um botão em uma página web faz parte do frontend, pois o usuário pode visualizá-lo e interagir com ele.
