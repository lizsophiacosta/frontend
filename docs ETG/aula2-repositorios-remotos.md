\# Conceitos de Git e Repositórios



\## Repositórios remotos

Locais externos (como GitHub ou GitLab) onde o código é armazenado e compartilhado.



\## Push

Envia alterações do repositório local para o remoto.



\## Pull

Traz atualizações do repositório remoto para o local.



\## Clone

Cria uma cópia completa de um repositório remoto no computador.



\## Branch

Ramificação do projeto que permite desenvolver novas funcionalidades sem afetar o código principal.



\## Merge

Integra alterações de uma branch ao código principal ou a outra branch.
