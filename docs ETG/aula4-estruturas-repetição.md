\# Estruturas de Repetição



\## O que são estruturas de repetição

Estruturas de repetição são usadas na programação para executar um bloco de código várias vezes, enquanto uma condição for verdadeira ou por um número determinado de vezes.



\## Como funcionam

O programa repete a execução de um trecho de código até que uma condição deixe de ser satisfeita ou até atingir um limite definido.



\## Principais tipos



\### for

Utilizado quando se sabe previamente quantas vezes o código deve ser repetido.



Exemplo:

for (let i = 0; i < 5; i++) {

&#x20;   // será executado 5 vezes

}



\### while

Executa o bloco de código enquanto a condição for verdadeira.



Exemplo:

while (contador < 5) {

&#x20;   // executa enquanto contador for menor que 5

}



\### do-while

Executa o bloco de código pelo menos uma vez, mesmo que a condição seja falsa.



Exemplo:

do {

&#x20;   // executa ao menos uma vez

} while (contador < 5);



\## Controle de repetição



\### break

Interrompe a execução do loop antes do fim.



\### continue

Pula para a próxima repetição do loop.



\## Cuidados importantes



\- Evitar loops infinitos (quando a condição nunca se torna falsa)

\- Garantir que a variável de controle seja atualizada corretamente



\## Resumo

Estruturas de repetição permitem automatizar tarefas repetitivas, tornando o código mais eficiente e organizado.
