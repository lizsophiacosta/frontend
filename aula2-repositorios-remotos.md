- Repositórios remotos: locais externos (como GitHub ou GitLab) onde o código é armazenado e compartilhado.

Push: envia alterações do repositório local para o remoto.

Pull: traz atualizações do repositório remoto para o local.

Clone: cria uma cópia completa de um repositório remoto no computador.

Branch: ramificação do projeto que permite desenvolver novas funcionalidades sem afetar o código principal.

Merge: integra alterações de uma branch ao código principal ou a outra branch.
